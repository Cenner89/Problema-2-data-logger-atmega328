## Tema

Projeto de um **Data-Logger Inteligente para Monitoramento Ambiental Remoto** usando o microcontrolador **ATmega328**.

## Resumo do que se pede

O sistema deve coletar medicoes ambientais por sensores analogicos e digitais, registrar data e hora usando um modulo RTC, aplicar um filtro digital passa-baixas aos sinais analogicos e disponibilizar os dados por comunicacao serial ou sem fio.

## Observacoes iniciais

- Diferente do Problema 1, este problema pede implementacao em linguagem C.
- O projeto deve ser pensado de forma modular.
- A documentacao sera uma parte importante da entrega.
- A entrega final indicada no enunciado e em 06/07/2026.
